# Coding Standards

- **TypeScript Strict Mode**: No `any` types unless absolutely necessary.
- **No Duplicated Logic**: Extract shared functionality into utilities.
- **Modular Architecture**: Keep systems decoupled.
- **Reusable Systems**: Build managers that can be reused across showcases.
- **SOLID Principles**: Adhere to Single Responsibility, Open-Closed, etc.
- **Dispose Resources**: Babylon meshes, textures, materials must be disposed of correctly.
- **Optimize Draw Calls & Memory**: Use instances, compress textures.
- **Document Public APIs**: Use JSDoc for all public methods and classes.
- **Strong Typing**: Enforce interfaces and types.
- **Professional Folder Naming**: `src/core`, `src/showcases`, `src/assets`.
