# UI Guidelines

- **Modern dark theme**: Professional UI with #000/#111 backgrounds and high-contrast text.
- **Professional spacing**: Use consistent padding and margins (e.g., 8px, 16px, 24px).
- **No overlapping UI**: Ensure UI components have dedicated screen real estate.
- **Responsive**: Scale correctly across Desktop, Tablet, and Mobile.
- **Accessibility**: Ensure high contrast, readable fonts, and ARIA tags if HTML-based.
- **Smooth animations**: Use GSAP or CSS transitions for UI elements.
- **Consistent typography**: Use a modern sans-serif font (e.g., Inter, Roboto).
- **Consistent button styles**: Use a unified design language for interactions.
- **Proper z-index rules**:
  - `z-index: 10` - Dedicated Debug layer
  - `z-index: 20` - Dedicated HUD layer
  - `z-index: 30` - Dedicated Overlay layer
