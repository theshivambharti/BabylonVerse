# Contributing to BabylonVerse

We welcome contributions! Please follow the `AGENTS.md` guidelines and read the `docs/CodingGuide.md` before submitting a Pull Request.

## Workflow
1. Fork the repository.
2. Create a feature branch (`feature/my-feature`).
3. Adhere strictly to the project's TypeScript and Babylon.js optimization rules.
4. Open a Pull Request for review.
