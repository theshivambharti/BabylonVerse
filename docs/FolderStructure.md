# Folder Structure

- `src/core/`: Core managers (App, Scene, Assets, Camera, Config, Environment, Lighting).
- `src/showcases/`: Isolated showcase modules (Materials, Models, Lighting, etc).
- `src/assets/`: Textures, HDR environments, GLB models.
- `src/scenes/`: (Deprecated/Legacy scenes - being migrated to showcases).
- `docs/`: Technical documentation.
- `.agents/`: AI OS rules and context.
